# Git Lab Task Manager 📝

**One-liner:** Мінімалістичний веб-застосунок для керування щоденними завданнями, створений для демонстрації навичок роботи з Git.

## 🛠 Стек технологій
* HTML5
* CSS3
* Vanilla JavaScript

## 🚀 Інструкція з розгортання
Оскільки це статичний веб-сайт без бекенду, для запуску достатньо:
1. Клонувати репозиторій: `git clone <посилання_на_ваш_репозиторій>`
2. Відкрити папку проєкту.
3. Відкрити файл `index.html` у будь-якому сучасному браузері.

## 📌 Статус
Розробляється в рамках Лабораторної роботи №1.
